# AGENTS.md

Прочитай главный индекс Memory Bank: [memory-bank/index.md](memory-bank/index.md)

Создавай новые ветки через:
git worktree add -b <branch> <path>
вместо git checkout -b.

После создания нового worktree всегда переходи в него и выполняй ./init.sh.
Любые PostgreSQL команды агент запускает только через ./bin/dip.
