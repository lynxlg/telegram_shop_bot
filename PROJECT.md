# Telegram-бот для интернет-магазина

- **Стек:** Aiogram (Python) 
- **База данных:** PostgreSQL

## Интерфейс и навигация

 - **Приветствие и главное меню:** Кнопки «Каталог», «Корзина», «Контакты», «Статус заказа».

- **Категории товаров:** Древовидная структура (например: Одежда -> Футболки -> Белая футболка).

- **Карточка товара:**

    - Фото/медиа (несколько изображений или видео).

    - Описание, цена, артикул.

    - Характеристики (размер, цвет, вес).

- **Поиск и фильтры:** Возможность найти товар по названию или фильтровать по цене/характеристикам.

## Корзина и оформление заказа

- **Добавление/удаление:** Кнопка «В корзину» в карточке товара. Возможность изменить количество или удалить позицию в корзине.

- **Сохранение корзины:** Данные должны сохраняться между сессиями (чтобы пользователь не потерял товары, если закроет бота).

- **Автоматическое оформление заказа:**

    - Сбор контактных данных (спросить номер телефона и адрес).

    - Подтверждение заказа: бот присылает сводку (что заказал, сумма) и просит нажать «Подтвердить».

    - Генерация номера заказа: Автоматическая нумерация (№1234).

## Оплата

- **Онлайн-оплата:** Интеграция с платежной системой

- **Оплата при получении**

- **Статус оплаты:** Бот должен менять статус заказа в базе данных (Ожидает оплаты -> Оплачен).

## Управление заказами

- **Трекинг:** Кнопка «Мои заказы» → статус (Принят, Собран, Передан в доставку, Получен).

- **Уведомления:** Автоматическая рассылка в чат покупателю при смене статуса заказа.

- **Отмена заказа:** Кнопка отмены (с комментарием причины).

## Административная панель

- **Роли:** Разделение прав 

- **Просмотр заказов:** Список новых заказов с кнопками для смены статуса.

- **Управление товарами:** Возможность добавлять/редактировать/удалять товары и категории прямо через Telegram

- **Массовые рассылки:** Отправка сообщений (текст/фото) всем пользователям бота.

- **Статистика:** Количество пользователей, выручка за день/неделю, популярные товары.

## Дополнительные функции 

- **Отзывы:** Запрос отзыва после получения заказа (кнопка «Оставить отзыв»).

- **Связь с поддержкой:** Кнопка «Оператор», которая перенаправляет сообщение менеджеру.
