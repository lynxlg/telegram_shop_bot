# Brief: Каталог товаров в Telegram-боте

  ## Проблема

  Сейчас пользователь интернет-магазина в Telegram-боте не может самостоятельно просматривать ассортимент товаров и получать информацию по конкретным позициям. 
  
  ## Для кого решаем

  - Основной пользователь: покупатель интернет-магазина, который хочет выбрать товар внутри Telegram-бота без обращения в поддержку.
  - Стейкхолдер: владелец магазина / менеджер продаж

  ## Контекст

  Проект развивается как Telegram-бот для интернет-магазина с дальнейшими сценариями корзины, оформления заказа и отслеживания статуса. Без каталога пользователь не может
  пройти базовый путь выбора товара, поэтому остальные коммерческие функции бота не дают ожидаемой ценности. Задача важна сейчас, потому что каталог является обязательной  основой для последующего оформления заказа внутри бота.

  ## Желаемый результат

  Пользователь может самостоятельно:

  - открыть список доступных товаров в боте;
  - перейти от общего ассортимента к нужной группе товаров;

  ## Критерий готовности

  Задача считается решённой, если пользователь без участия менеджера может пройти путь:

  1. открыть каталог;
  2. найти нужный раздел ассортимента;
  3. открыть товар;
  4. получить название, цену, описание и основные характеристики товара, достаточные для перехода к следующему шагу заказа
