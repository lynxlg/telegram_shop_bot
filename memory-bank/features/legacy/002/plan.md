- Для обеих веток показа карточки переиспользовать текущий `build_product_text()` из [app/services/catalog_text.py](/home/lynx/telegram_shop_bot/app/services/catalog_text.py) без изменения этого файла.

## Критерий готовности

Фича считается готовой, когда:

- у модели `Product` и схемы БД есть поле `image_url`;
- в seed-данных есть как минимум один товар с `image_url` и один товар без `image_url`;
- пользователь видит карточку товара с изображением, если `image_url` задан;
- пользователь видит карточку товара без изображения, если `image_url` пустой или `NULL`;
- карточка товара сохраняет текущие fallback для `description` и `attributes`;
- кнопка `Назад` возвращает на предыдущий экран каталога;
- ошибка показа карточки с изображением приводит к сообщению `Не удалось загрузить каталог. Попробуйте позже.` и записи ошибки в лог;
- миграция и тесты проходят.
